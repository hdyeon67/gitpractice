# 📘 Git Learning Log

이 저장소는  
**비즈니스 문제 해결을 위한 실무형 AI·데이터 분석 과정**을 수강하며  
Git과 GitHub의 기본 개념과 협업 흐름을 학습·정리하기 위한 공간입니다.

현재는 Git을 시작 단계로 학습하고 있으며,  
향후 데이터 분석 및 프로젝트 기록으로 확장할 예정입니다.

---

## 🎯 학습 목적

- Git의 기본 개념을 이해하고  
  단순한 명령어 사용이 아니라 **변경 이력과 협업 흐름을 이해**
- GitHub를 활용한 **버전 관리 습관** 형성
- README를 통해 학습 의도와 과정을 **명확히 문서화**

---

## 🧭 학습 방향과 각오

> Git은 도구가 아니라  
> 협업과 기록을 위한 언어라고 생각합니다.

- 명령어 암기보다 **왜 사용하는지 이해**
- 작은 변경도 의미 있는 단위로 커밋
- 커밋 메시지에 **의도 남기기**
- 학습 과정 자체를 하나의 결과물로 관리

---

## 🧩 학습 내용 (Git)

### ✔ 기본 개념
- Git과 GitHub의 역할 차이
- Repository, Commit, Branch 개념
- 로컬 저장소와 원격 저장소의 관계

### ✔ 기본 명령어
- `git init`
- `git status`
- `git add`
- `git commit`
- `git log`
- `git push`
- `git pull`
